# 更新

## 更新命令行工具

::: code-group

```bash [npm]
npm i -g @ddd-tool/domain-designer-cli@latest
```

```bash [pnpm]
pnpm i -g @ddd-tool/domain-designer-cli@latest
```

```bash [bun]
bun i -g @ddd-tool/domain-designer-cli@latest
```

:::

## 更新工作空间

切换到工作空间的根目录，执行命令：

```bash
domain-designer-cli update
```
